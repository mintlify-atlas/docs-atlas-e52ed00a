## Overview

The `Navbar` component is the primary navigation bar displayed at the top of every page. It includes the application logo, language selector, FAQ link, GitHub link, and a compress action button. The component supports internationalization and dark mode.

## Props

<ParamField path="locale" type="string" default="en">
  The current locale for translations (e.g., 'en', 'es', 'hi', 'ar')
</ParamField>

## Key Features

- **Responsive Design**: Adapts to mobile and desktop screens with conditional rendering
- **Dark Mode Support**: Automatic dark mode styling with Tailwind classes
- **Internationalization**: Uses translation system via `getPageTranslations`
- **Logo Display**: Shows the MC World Compressor logo with Next.js Image optimization
- **Language Selection**: Integrates the `LanguageSelector` component
- **Quick Actions**: FAQ navigation and GitHub repository link
- **Primary CTA**: Prominent "Compress" button to navigate to upload page

## Usage Example

```jsx
import Navbar from '@/components/Navbar';

export default function Layout({ params }) {
  const locale = params.locale || 'en';
  
  return (
    <div>
      <Navbar locale={locale} />
      {/* Page content */}
    </div>
  );
}
```

## Translation Keys

The component uses the following translation keys from the `nav` namespace:

- `t.compress` - Text for the compress button (varies by locale)

## Styling

The Navbar uses Tailwind CSS utility classes for:
- Responsive padding and spacing
- Dark mode color schemes
- Smooth hover transitions
- Flexbox layout for alignment
- Border styling for visual separation

## Dependencies

- `next/link` - Client-side navigation
- `next/image` - Optimized image loading
- `@/components/LanguageSelector` - Language switching component
- `@/lib/translations` - Translation utilities
